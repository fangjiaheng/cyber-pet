## 一、基础属性

### 1. 饱食度（hunger）

| 项目 | 数值 |
|------|------|
| 范围 | 0 ~ `3000 + 100 × min(level, 30)`，最大 **6000** |
| 初始值 | 上限 × 80% |
| 每分钟衰减 | -5 ~ -8（随机） |
| 心情 < 600 时 | 额外衰减 -2（即 -7 ~ -10） |

**不适阈值：**
- `< 720` → 饥饿不适 → 可触发肚子胀链
- `> 上限 - 260` → 饱食不适 → 可触发感冒链

### 2. 清洁度（cleanliness）

| 项目 | 数值 |
|------|------|
| 范围 | 0 ~ `3000 + 100 × min(level, 30)`，最大 **6000** |
| 初始值 | 上限 × 80% |
| 每分钟衰减 | -5 ~ -8（随机） |
| 心情 < 600 时 | 额外衰减 -2（即 -7 ~ -10） |

**不适阈值：**
- `< 1080` → 脏乱不适 → 可触发咳嗽链

### 3. 心情值（mood）

| 项目 | 数值 |
|------|------|
| 范围 | 0 ~ **1000** |
| 初始值 | 800 |
| 每分钟衰减 | -2 ~ -4（随机） |

**心情外观映射（决定动画表现）：**

| 心情值 | 外观状态 | 说明 |
|--------|---------|------|
| ≥ 900 | `happy` | 开心 |
| ≥ 800 | `peaceful` | 平静 |
| ≥ 500 | `upset` | 沮丧 |
| < 500 | `sad` | 心情差 |

**心情对成长的惩罚：**

| 心情值 | 成长惩罚 |
|--------|---------|
| ≥ 900 | 0 |
| ≥ 700 | -20 |
| ≥ 500 | -70 |
| ≥ 300 | -100 |
| ≥ 100 | -140 |
| < 100 | -180 |

### 4. 能量值（energy）

| 项目 | 数值 |
|------|------|
| 范围 | 0 ~ **100** |
| 初始值 | 80 |
| 每分钟衰减 | -0.2 |
| 睡觉时 | 每分钟 +1 |

### 5. 健康值（health）

| 项目 | 数值 |
|------|------|
| 范围 | 0 ~ **5** |
| 初始值 | 5（健康） |
| 计算方式 | `5 - 疾病阶段`（无病=5，阶段1=4 ... 阶段4=1） |
| 健康 = 0 | 死亡，停止成长 |

**健康对成长的惩罚：**

| 健康值 | 成长惩罚 |
|--------|---------|
| 5 | 0 |
| 4 | -30 |
| 3 | -80 |
| 2 | -120 |
| 1 | -160 |
| 0 | 停止成长 |

## 四、疾病系统

### 触发条件
不适类型连续累计 **> 4 次**（每分钟检测一次），触发对应疾病链。

### 三条疾病链

**感冒链（饱食不适触发：饱食度 > 上限-260）**

| 阶段 | 疾病 | 健康值 | 对症药品 |
|------|------|--------|---------|
| 1 | 感冒 | 4 | 板蓝根 (10001) |
| 2 | 发烧 | 3 | 退烧药 (30004) |
| 3 | 重感冒 | 2 | 银翘丸 (20001) |
| 4 | 肺炎 | 1 | 金色消炎水 (30001) |

**咳嗽链（脏乱不适触发：清洁度 < 1080）**

| 阶段 | 疾病 | 健康值 | 对症药品 |
|------|------|--------|---------|
| 1 | 咳嗽 | 4 | 枇杷糖浆 (10003) |
| 2 | 支气管炎 | 3 | 甘草剂 (20003) |
| 3 | 哮喘 | 2 | 定喘丸 (30003) |
| 4 | 肺结核 | 1 | 通风散 (40003) |

**肚子胀链（饥饿不适触发：饱食度 < 720）**

| 阶段 | 疾病 | 健康值 | 对症药品 |
|------|------|--------|---------|
| 1 | 肚子胀 | 4 | 消食片 (10002) |
| 2 | 胃炎 | 3 | 蓝色消炎水 (20002) |
| 3 | 胃溃疡 | 2 | 龙胆草 (30002) |
| 4 | 胃癌 | 1 | 仙人汤 (40002) |

**万能药品：**
- 百草丹 (50001)：降低一个疾病阶段
- 还魂丹 (60001)：直接痊愈

**恶化规则：** 未治疗的疾病每 **10 分钟** 阶段 +1

## 五、成长系统

### 成长阶段

| 等级 | 阶段 |
|------|------|
| 1 ~ 4 | 蛋（egg） |
| 5 ~ 8 | 幼年（kid） |
| ≥ 9 | 成年（adult） |

### 每分钟成长值计算

```
成长值 = max(0, 260 - 心情惩罚 - 健康惩罚 - 属性惩罚)
```

**属性惩罚：**
- 饱食度 < 720 → -80
- 清洁度 < 1080 → -80
