# 🎬 在企鹅身上播放 SWF 动画

## ✅ 已完成的功能

现在你可以直接在企鹅身上播放 Flash SWF 动画了！

### 功能特点

1. **直接替换显示**：播放 SWF 时，Flash 动画会替换企鹅的 PNG 显示
2. **一键播放**：在动画画廊选择动画后，点击"在企鹅上播放"按钮
3. **一键停止**：播放时显示停止按钮⏹️，随时可以停止
4. **自动循环**：SWF 动画自动循环播放
5. **无缝切换**：停止后立即恢复 PNG 企鹅显示

---

## 🚀 使用方法

### 步骤 1：启动项目

```bash
npm run dev
```

### 步骤 2：打开动画画廊

1. 鼠标移到企鹅上
2. 点击 **🎬 按钮**（动画画廊）

### 步骤 3：选择并播放动画

1. 在画廊中选择任意分类（如"常规动作"）
2. 点击任意动画（如"动作1"）
3. 看到画廊底部的预览播放器
4. 点击 **"🐧 在企鹅上播放"** 按钮

### 步骤 4：在企鹅上观看

- 画廊会自动关闭
- 企鹅的位置显示 Flash 动画
- 动画自动循环播放
- 显示气泡提示："正在播放动画..."

### 步骤 5：停止播放

- 点击动画下方的 **⏹️ 停止播放** 按钮
- 企鹅立即恢复 PNG 显示
- 回到待机状态

## 🔧 技术实现

### 修改的文件

1. **App.tsx**
   - 添加 `currentSwfUrl` 状态
   - 添加 `handlePlaySwf` 和 `handleStopSwf` 函数
   - 在企鹅渲染区域添加条件渲染：SWF 或 PNG
   - 传递 `onPlaySwf` 回调给 SwfGallery

2. **SwfGallery.tsx**
   - 添加 `onPlaySwf` 属性接口
   - 添加"在企鹅上播放"按钮
   - 点击按钮时调用 `onPlaySwf(selectedSwf)`

3. **SwfGallery.css**
   - 添加 `.play-on-pet-btn` 按钮样式

4. **App.css**
   - 添加 `.swf-player-container` 容器样式
   - 添加 `.stop-swf-btn` 停止按钮样式

### 核心代码逻辑

```typescript
// App.tsx
const [currentSwfUrl, setCurrentSwfUrl] = useState<string | null>(null)

const handlePlaySwf = (swfUrl: string) => {
  setCurrentSwfUrl(swfUrl)      // 设置要播放的 SWF
  setShowSwfGallery(false)      // 关闭画廊
}

const handleStopSwf = () => {
  setCurrentSwfUrl(null)        // 清空 SWF
  setPenguinAction('idle')      // 恢复待机
}

// 条件渲染
{currentSwfUrl ? (
  <RufflePlayer src={currentSwfUrl} />
) : (
  <QQPenguinSprite action={penguinAction} />
)}
```

## 🎨 动画效果对比

### PNG 企鹅（默认）
- ✅ 体积小（每个动作 ~100KB）
- ✅ 加载快
- ✅ 流畅 60fps
- ❌ 动画帧数少（~25帧）
- ❌ 没有音效

### SWF Flash 动画
- ✅ 原版动画（完整帧数）
- ✅ 包含音效
- ✅ 矢量图形（无损缩放）
- ❌ 体积较大（需要 Ruffle）
- ❌ 首次加载较慢

## 💡 使用建议

### 推荐场景

**日常使用：PNG 动画**
- 待机、吃饭、洗澡等高频动作

**特殊展示：SWF 动画**

### 最佳实践

1. **预览后播放**
   - 先在画廊中预览
   - 确认是想要的动画
   - 再点击"在企鹅上播放"

2. **及时停止**
   - SWF 播放完一次后会自动循环
   - 看完后记得点停止按钮
   - 避免长时间占用资源

3. **切换动画**
   - 想看另一个动画时
   - 先停止当前 SWF
   - 再打开画廊选择新动画

## 🔍 常见问题

### Q: 为什么点击"在企鹅上播放"后没反应？

A: 检查以下几点：
1. 确认 Ruffle 文件已复制到 `public/ruffle/`
2. 查看浏览器控制台是否有错误
3. 确认 SWF 文件路径正确

### Q: 动画播放卡顿怎么办？

A:
- SWF 动画首次加载需要初始化 Ruffle
- 等待 2-3 秒后会流畅播放
- 如果持续卡顿，可能是文件损坏

### Q: 停止播放后企鹅没有恢复？

A:
- 刷新页面重试
- 检查控制台是否有 JavaScript 错误

### Q: 能同时播放多个 SWF 吗？

A:
- 不能，一次只能播放一个
- 点击新动画会自动替换当前动画

### Q: SWF 动画有音效吗？

A:
- 是的！原版 Flash 动画包含音效
- 确保浏览器没有静音
- 首次播放可能需要点击允许音频
