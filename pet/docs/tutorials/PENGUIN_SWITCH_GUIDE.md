# 🎨✨ 企鹅双版本切换指南

## ✅ 已实现功能

### 1️⃣ 两个版本可切换
- **CSS版本** 🎨 - 用CSS绘制的企鹅（轻量、即时加载）
- **原版素材** ✨ - QQ宠物真实素材（100%还原）

### 2️⃣ 两种企鹅可选择
- **GG** 🐧 - 哥哥/男性企鹅（蓝色）
- **MM** 🎀 - 妹妹/女性企鹅（粉色）

### 3️⃣ 动态切换按钮
- 渐变色圆形按钮
- 悬停动画效果
- 智能显示/隐藏

---

## 🎮 使用方法

### 启动项目
```bash
npm run dev
```

### 切换按钮位置
在企鹅下方会看到切换按钮：

```
    🐧 (企鹅)
    ↓
  [ 🎨 ][ 🐧 ]  ← 切换按钮
```

### 按钮功能

#### 🎨/✨ 版本切换按钮（橙色渐变）
- **点击** → 在CSS版本和原版素材之间切换
- **🎨** = 当前是原版素材，点击切换到CSS版本
- **✨** = 当前是CSS版本，点击切换到原版素材

#### 🐧/🎀 类型切换按钮（蓝绿色渐变）
- **仅在原版素材模式下显示**
- **点击** → 在GG和MM之间切换
- **🐧** = 当前是GG（哥哥），点击切换到MM
- **🎀** = 当前是MM（妹妹），点击切换到GG

## 🎯 使用场景

### 场景1：查看两个版本的对比

1. 启动项目，默认显示 **CSS版本**
2. 点击 **✨ 按钮**，切换到原版素材
3. 再次点击 **🎨 按钮**，切换回CSS版本
4. 对比两个版本的动画效果

### 场景2：切换企鹅性别

1. 点击 **✨ 按钮**，切换到原版素材
2. 此时会出现 **🐧 按钮**
3. 点击 **🐧/🎀 按钮**，切换企鹅类型
4. GG 和 MM 的动画略有不同

### 场景3：测试不同动作

```
1. 悬停在企鹅上 → 显示功能按钮
2. 点击 🍖 喂食 → 看企鹅吃东西动画
3. 切换版本 → 对比CSS版和原版的吃东西动画
4. 切换类型 → 对比GG和MM的吃东西动画
```

## 🔧 自定义

### 修改默认版本

编辑 `App.tsx`：

```tsx
// 默认使用原版素材
const [useRealSprite, setUseRealSprite] = useState(true)

// 默认使用MM
const [penguinType, setPenguinType] = useState<PenguinType>('MM')
```

编辑 `App.css`：

```css
.control-btn {
  width: 50px;     /* 改变大小 */
  height: 50px;
  font-size: 24px; /* 改变图标大小 */
}
```

### 添加更多切换选项

在 `App.tsx` 中添加新的状态和按钮。

## 🐛 常见问题

### Q1: 切换按钮不显示？

**检查**：
- 确认 App.tsx 有 `penguin-controls` div
- 查看浏览器控制台是否有错误

### Q2: 切换到原版后一直显示"加载中..."？

**原因**：素材未正确复制

**解决**：
```bash
# 检查素材是否存在
ls public/assets/penguin_original/GG/

# 如果没有，重新复制
cd /tmp/QQPet13
cp -r res/atlas/ui/ani/* /path/to/your/project/public/assets/penguin_original/
```

### Q3: 切换类型按钮一直不出现？

**原因**：没有切换到原版素材

**解决**：
1. 先点击 ✨ 切换到原版
2. 然后 🐧/🎀 按钮会出现

### Q4: 想隐藏切换按钮？

在 `App.css` 中添加：
```css
.penguin-controls {
  display: none;  /* 隐藏所有切换按钮 */
}
```
