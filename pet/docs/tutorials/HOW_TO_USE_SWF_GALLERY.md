## 🚀 如何使用

### 步骤 1：启动项目

在终端运行：
```bash
cd /Users/erase/Desktop/react_Project/pet
npm run dev
```

### 步骤 2：打开动画画廊

1. 等待项目启动完成
2. 鼠标**悬停在企鹅上**，功能按钮会显示出来
3. 点击 **🎬 按钮**（第三个按钮）
4. 动画画廊会弹出！

### 步骤 3：浏览动画

画廊界面有三个部分：

**1. 顶部分类标签**
- 🐧 常规动作 (17个)
- 🍖 饮食活动 (9个)
- 🤒 生病状态 (3个)
- ✨ 特殊动作 (9个)
- 💭 状态 (2个)

**2. 中间动画列表**
- 显示当前分类下的所有动画
- 点击任意动画名称即可预览

**3. 底部播放器**
- 实时播放选中的 Flash 动画
- 自动循环播放

### 步骤 4：关闭画廊

点击右上角的 **✕** 按钮即可关闭

## 💡 提示

### 如果画廊没有显示
1. 确认鼠标已经悬停在企鹅上（功能按钮必须先显示）
2. 确认点击的是 🎬 按钮（第三个）
3. 检查浏览器控制台是否有错误（F12）

### 如果动画不播放
1. 确认 Ruffle 已正确加载（首次加载需要几秒）
2. 检查 .swf 文件路径是否正确
3. 查看浏览器控制台错误信息

### 如果按钮不显示
- 功能按钮只在**鼠标悬停在企鹅上**时显示
- 移动鼠标离开后会在 3 秒后自动隐藏
