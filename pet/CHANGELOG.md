# 更新日志

所有重要变更都会记录在这里。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [0.5.1] - 2026-03-15

### 变更

- 功能栏下方不再显示等级、经验、元宝、心情等状态摘要，浮层信息更简洁
- 喂食和清洁横向滚动条改为跟随功能栏定位，出现在功能区正下方

### 修复

- 移除启动阶段的企鹅占位 fallback，避免初始化时出现风格不一致的临时图形

## [0.5.0] - 2026-03-14

### 新增

- 新增 `HorizontalScrollStrip` 横向滚动选择组件：圆形图标项、左右箭头翻页、hover tooltip、标题栏、关闭按钮
- 喂食功能改为横向滚动条选择，合并动画分类中的饮食项，共 10 种食物
- 清洁功能改为横向滚动条选择，合并清洁用品，共 6 种
- 互动菜单中点击"喂食"或"清洁"会关闭下拉菜单并展开对应横向滚动条
- 右键菜单中"喂食"和"清洁"入口同样切换到横向滚动条

### 变更

- `swfData.ts` 移除"饮食"分类（食物和清洁用品已拆分到各自的横向滚动条），动画菜单从 9 个分类降为 8 个
- 横向滚动条不再使用 scrollbar，改为左右箭头按钮 + 隐藏滚动条

### 修复

- 修复企鹅顶部留白问题（CSS padding 压缩）

## [0.4.0] - 2026-03-13

### 新增

- 新增 `AI 助手配置` 面板，可查看当前 Base URL、API Key 掩码、默认模型和配置状态
- 新增可复用 `ActionDropdownMenu` 组件，用于喂食菜单和动画菜单
- 新增 `player.swf` 控制桥验证面板，开发态可直接检查 `loadlists` / `setid`
- 新增 `penguin102OriginalPlaylists.ts`，保存 102 号企鹅无法由默认规则推导的原版特殊 playlist

### 变更

- 正式播放器从“每次重新加载单个动作 SWF”切换为“常驻 `/player.swf` + `loadlists` 切动作”
- `buildLoadlistsPlaylist()` 现在支持按 `animationId` 优先命中原版完整动作列表
- 动画菜单和喂食菜单统一把原版动作 id 透传给播放链路
- 设置入口名称统一为 `AI 助手配置`
- 主渲染入口移除了 `StrictMode`，避免开发态 Ruffle 双挂载干扰

### 修复

- 修复动画切换时的整屏闪烁问题
- 修复“看书 / 来信 / 记笔记 / 瞌睡”等多段动作播放不完整的问题
- 修复右键菜单首次以外无法可靠外部关闭的问题
- 修复右键菜单展开时位置偏移过大、遮挡宠物过多的问题
- 修复功能区下拉菜单受外部容器裁切的问题
- 修复 AI 聊天窗口和设置面板拖动异常的问题
- 修复气泡文本和部分长内容被容器裁切的问题
- 修复下拉菜单出现横向滚动条的样式问题

### 文档

- 更新 `README.md`、`PROJECT_STATUS.md`、`TODO.md`、`docs/README.md`、`docs/dev/PROGRESS.md`
- 补充“当前版本仍属原版素材验证原型，商业化前需原创化替换”的边界说明
