### 2. SWF 切换闪烁 🐛
**任务**：消除切换动画时的短暂空白闪烁
**根本原因**：Ruffle 使用 WebAssembly + Worker 渲染，`load()` 期间 canvas 强制清空，`toDataURL` 截帧无法在正确时机拿到当前帧。这是 Ruffle 底层限制。
**已尝试方案**（均无效）：
- 单例模式（不销毁实例）
- 截帧覆盖（`toDataURL` 时机不可靠）
- 防抖延迟切换
- 双缓冲

**可能的解法**：
- [ ] 等待 Ruffle 官方支持 `preload()` API
- [ ] 研究是否可以用 `gotoAndPlay()` 在单个 SWF 内切换动作（需要找到支持多动作的主 SWF）
- [ ] 用 Canvas 2D 自行实现帧缓冲层，在 Ruffle canvas 上方叠加

**优先级**：🔥 高
**状态**：阻塞，暂缓
