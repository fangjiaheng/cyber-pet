# Q宠宝贝

基于 Electron + React + Ruffle 的桌面宠物原型项目，致敬经典 QQ 宠物。

当前版本仍使用原版 QQ 宠物相关 SWF 资源做交互与动画验证，因此只适合本地学习、原型验证和技术研究，不适合直接公开分发或商业化。若要长期发布，必须先完成角色、美术、命名、音效与文案的原创化替换。

## 当前能力

- 桌面宠物常驻显示，支持拖拽、悬浮、右键菜单和透明区域穿透
- Ruffle 常驻加载 `player.swf`，通过 `setid()` + `loadlists()` 切换动作，减少整屏闪烁
- 功能区支持动画、日常、交互、任务等入口，支持 click 和 hover 触发下拉菜单
- 完整养成系统：动态属性上限、渐进等级经验表、成长阶段、疾病系统
- 物品库存系统：物品消耗型喂食/清洁/治疗，库存持久化
- 商店系统：4分类商品（食物/清洁/药品/背景），元宝购买
- 打工系统：18种工作，等级/学历门槛，元宝+属性奖励
- 学习系统：9科目×4学校等级（小学/中学/大学/研究生），学时追踪
- 旅行系统：4个旅行目的地，心情/属性奖励
- 对话系统：200+条原版对话，定时闲聊、状态提醒、事件触发
- 疾病系统：3条疾病链×4级，对症药品匹配，万能药品支持
- 信息面板：宠物资料卡、状态进度面板、背包浏览
- 阶段转变动画：蛋→幼年、幼年→成年过渡动画
- 系统托盘：根据宠物状态切换托盘图标动画
- AI 聊天窗口、AI 配置面板、本地聊天记录与设置持久化
- 任务系统：登录送礼 / 在线送礼，支持本地持久化与重置

## 系统详细说明

### 物品与商店

- 完整物品目录：50+食物、22清洁用品、22药品、17背景
- 喂食/清洁/治疗消耗库存中的物品，库存不足时提示去商店
- 商店面板 4 个分类标签，分页浏览，元宝支付购买
- 新建存档自动发放入门物品（食物、清洁品、基础药品）

### 打工与学习

- 打工：18种工作（搬砖→公务员），等级和学历门槛，30-120分钟计时，元宝+属性奖励
- 学习：9门科目（语文/数学/政治/音乐/艺术/礼仪/体育/劳技/武术），4个学校等级
- 活动互斥：打工/学习/旅行期间禁止其他活动
- 学历影响打工工种：高级工作需要对应科目达到一定学时

### 对话系统

- 启动时随机显示进入对话
- 每4分钟自动显示闲聊对话
- 饥饿/脏乱时优先显示状态提醒
- 喂食/清洁/升级后触发对应事件对话
- 所有对话自动替换主人名占位符

### 疾病与治疗

- 3条疾病链，每条4级（如：感冒→发烧→重感冒→肺炎）
- 每种疾病有对应的对症药品
- 治疗条智能标注对症药品，使用错误药品时提示
- 百草丹治愈一级，还魂丹直接痊愈
- 健康值 0-5 级，影响成长速度

### 信息面板

- 宠物资料卡：名称（可编辑）、属性、等级、元宝、各科学历
- 状态信息面板：饥饿/清洁/心情/体力/健康进度条、经验进度、在线时长
- 背包面板：按分类浏览已有物品，数量展示

### 任务系统

- 入口位于 `任务`
- 登录送礼按周期发放，共 12 格
- 在线送礼按在线分钟数解锁，阈值为 10 / 40 / 70 / 100 / 130 / 160 / 190 / 220 分钟
- 每天早上 `06:00` 刷新在线送礼周期
- 任务状态保存在本地 `electron-store`

### 动画系统

- 使用原版 SWF 资源作为当前验证素材
- 成长阶段感知：蛋/幼年/成年各阶段动态加载对应 SWF
- 心情外观切换：成年阶段 happy/peaceful/upset/sad 四种待机
- 阶段转变动画：蛋→幼年（Etoj.swf）、幼年→成年（Jtoc.swf）
- 系统托盘图标根据宠物状态切换动画帧

### 桌面交互

- 右键菜单：AI助手 / 宠物互动 / 商城 / 宠物信息 / 选项
- 宠物信息子菜单：宠物资料、宠物状态、背包
- 喂食和清洁使用横向滚动选择组件
- 气泡提示显示在宠物头部上方

### AI 助手

- 内置聊天窗口，支持流式回复
- 支持本地保存聊天历史
- AI 配置面板支持配置 Base URL、API Key、默认模型
- 支持测试连接和保存后即时生效

## 技术方案

### 动画播放

当前播放链路不再为每个动作单独重建播放器，而是：

1. 常驻加载 `public/player.swf`
2. 使用 `setid()` 指定角色
3. 使用 `loadlists()` 切换播放列表
4. 在 `src/renderer/utils/swfPlaylist.ts` 中做业务动作到原始 playlist 的映射

### 窗口布局

- `pet` 与 `bubble` 模式统一使用相同的锚点计算
- 主窗口是透明无边框 Electron 窗口，默认尺寸 `360x420`
- 渲染层通过 `resizeWindow` 在宠物、菜单、面板之间切换窗口尺寸

### 成长与状态

- 成长配置在 `src/renderer/stores/growthConfig.ts`
- 成长引擎在 `src/renderer/stores/growthEngine.ts`
- 疾病系统在 `src/renderer/stores/diseaseSystem.ts`
- 活动系统在 `src/renderer/stores/activitySystem.ts`
- 物品目录在 `src/shared/itemCatalog.ts`
- 库存管理在 `src/renderer/stores/inventoryStore.ts`

### 本地存储

`electron-store` 当前持久化以下内容：

- 宠物状态与养成数据（含成长阶段、疾病状态、存档版本）
- 物品库存
- 活动进度与学习学时
- 登录送礼 / 在线送礼状态
- AI 配置
- 聊天记录
- Token 统计记录
